# UBVoting

Running locally
---------------

1. Clone this repo:
	```sh
	git@github.com:bobmshannon/UBVoting.git
	```

2. Start the virtual machine:
	```sh
	vagrant up
	```

3. SSH in to the virtual machine:
	```sh
	vagrant ssh
	```

4. Access the the server locally at [http://localhost:8080/](http://localhost:8080/)
